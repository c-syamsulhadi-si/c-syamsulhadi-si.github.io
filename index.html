bootstrap

Bootstrap adalah framework open-source khusus front end yang awalnya dibuat oleh Mark Otto dan Jacob Thornton untuk mempermudah dan mempercepat pengembangan web di front end.

codeigniter

CodeIgniter merupakan framework dengan konsep MVC (Model, View, Controller). Framework ini dirilis pertama kali pada 28 Februari 2006 hingga sekarang masih dikembangkan baru-baru ini kembali hadir codeigniter versi 4.1.2 yang dirilis pada tanggal 18 Mei 2021

laravel
